Navigation Page

This page allows the user to move to different modules of the web application.


1. Personal Info Module


Change login information: you can change your user name and password.


Change personal information: you can change your first name, last name, address, Email, phone number.


2. User Management Module


Modify users: update an existing user's information, or create a new user using existing person's information.


Modify persons: similar to modifying users, you can either update or create a new person's information.


Modify doctors: update or create a doctor-patient relationship.


3. Report Generating Module


Generate reports with a specified diagnosis for a given time period. The list contains the name, address and phone number of the patient, and testing date of the first radiology record that contains the specified diagnosis.


4. Uploading Module


Radiology record: create a new radiology record by entering detailed information of a test, including Patient ID, Doctor ID, Radiologist ID, Test Type, Prescribing Date, Test Date, Diagnosis, Description.


Medical picture: you can upload a picture associate with a certain radiology record, by providing record ID.


5. Search Module


Search the database for a list of relevant radiology records and to view medical images with the zoom-in facility. Records displayed contain all the information, including the list of thumbnails of medical images, of the record. You can also choose one of four ways to sort the records.


6. Create a view for data analysis.


Create a view for data analysis.


7. Data Analysis Module


Generate and display an OLAP report for data analysis. You can choose to include patient name and test type, and time period by weekly, monthly, yearly.


8. Logout


Log out the current account and return to login page.
