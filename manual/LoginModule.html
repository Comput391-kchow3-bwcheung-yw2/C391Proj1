Login Module

This module lets the user login to the web application and change their user info.


Logging In:


1. In the login.html page enter user name and password and press login.


2. Redirect to login result and press redirect if successful or retry if unsuccessful.


3. User ends up on navigation home page.


Changing Login Info:


1. On navigation home page user selects Personal Info Module


2. Redirected to page that prompts user to select change login info.


3. User sees current info and can change the login credentials and presses submit.


4. User is redirected back and login info is changed.


Changing Personal Info:


1. On navigation home page user selects Personal Info Module


2. Redirected to page that prompts user to select change personal info.


3. User sees current info and can change personal info and presses submit.


4. User is redirected back and personal info is changed.
