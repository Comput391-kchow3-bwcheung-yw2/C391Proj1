Installation

To install the web application follow the steps below.


1. Extract the archive file containing the project.


2. Move all html and php files to the webdocs folder

(ex. /compsci/webdocs/<yourccid>/web_docs/)


3. In terminal of folder /compsci/webdocs/<yourccid>/web_docs set the permission by

the command "chmod 644 *"


4. Run the sql script "createindex.sql"


5. Run the sql script  "drjobdml.sql" and enter the index names and interval second for auto update.


6. Now go to login page and login to access the web application

(http://consort.cs.ualberta.ca/~<yourccid>/login.html)
