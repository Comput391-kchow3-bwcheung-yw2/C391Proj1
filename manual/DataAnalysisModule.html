Data Analysis Module


Generate and display an OLAP report for data analysis. You can choose to include patient name or test type by checking the boxes, and time period by weekly, monthly, yearly.


1. Create a view first by clicking create view in navigation page.


2. Click data analysis module.


3. Select options for the data analysis.


4. Redirected to result of the data analysis.
